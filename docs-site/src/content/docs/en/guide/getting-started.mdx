## Installation

### Prerequisites

- **Node.js 18+**
- **npm 9+** or **pnpm 8+**
- (Optional) Claude API key (for `RemoteLLMProvider`)

### Clone and install

```bash
git clone https://github.com/hyperskills/webmcp-auto-ui.git
cd webmcp-auto-ui

# Install dependencies (Turbo monorepo)
npm install
# or
pnpm install
```

### Monorepo structure

```
webmcp-auto-ui/
├── packages/
│   ├── core/          # WebMCP polyfill, MCP client, WebMCP server
│   ├── agent/         # Agent loop, tool layers, system prompt
│   ├── ui/            # 30+ Svelte widgets
│   └── sdk/           # Canvas store, HyperSkill encoding
├── apps/
│   ├── flex/          # Online demo (agent chat + widgets)
│   ├── viewer/        # HyperSkill reader
│   ├── home/          # Landing page
│   └── showcase/      # Widget gallery
└── scripts/
    └── deploy.sh      # Deploy to server
```

## Quick setup

### Example 1 : Simple agent loop

```typescript
import { runAgentLoop } from '@webmcp-auto-ui/agent';
import { RemoteLLMProvider } from '@webmcp-auto-ui/agent';

const provider = new RemoteLLMProvider({
  apiKey: process.env.ANTHROPIC_API_KEY,
  model: 'claude-3-5-sonnet-20241022',
});

const result = await runAgentLoop('Show me a chart of Q1-Q4 sales', {
  provider,
  maxIterations: 5,
});

console.log(result.text); // Final assistant text
console.log(result.toolCalls); // Tool calls made
```

### Example 2 : With WebMCP server (widgets)

```typescript
import { createWebMcpServer } from '@webmcp-auto-ui/core';
import { autoui } from '@webmcp-auto-ui/agent';

// autoui already contains 30+ native widgets
const layer = autoui.layer();

const result = await runAgentLoop('Show me stats', {
  provider,
  layers: [layer],
  maxIterations: 5,
});
```

### Example 3 : With MCP client (data)

```typescript
import { McpClient } from '@webmcp-auto-ui/core';

const mcpClient = new McpClient({
  url: 'http://localhost:3001',
  clientName: 'webmcp-auto-ui-demo',
});

await mcpClient.initialize();

const result = await runAgentLoop('List recipes', {
  provider,
  client: mcpClient,
  layers: [
    // ... your MCP and WebMCP layers
  ],
});
```

## First launch

### Run the local demo

```bash
# Build all packages
npm run build

# Launch flex app (agent chat + canvas)
cd apps/flex
npm run dev
```

The app is accessible at `http://localhost:5173`.

### Configure Claude API key (optional)

```bash
# In apps/flex/.env.local
VITE_ANTHROPIC_API_KEY=sk-ant-...
```

Without a key, you can use Gemma (in-browser WASM).
