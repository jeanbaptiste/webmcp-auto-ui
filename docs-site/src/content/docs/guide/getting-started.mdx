## Installation

### Prérequis

- **Node.js 18+**
- **npm 9+** ou **pnpm 8+**
- (Optionnel) Clé API Claude (pour `RemoteLLMProvider`)

### Cloner et installer

```bash
git clone https://github.com/hyperskills/webmcp-auto-ui.git
cd webmcp-auto-ui

# Installer les dépendances (monorepo Turbo)
npm install
# ou
pnpm install
```

### Structure du monorepo

```
webmcp-auto-ui/
├── packages/
│   ├── core/          # Polyfill WebMCP, client MCP, serveur WebMCP
│   ├── agent/         # Boucle agent, tool layers, system prompt
│   ├── ui/            # 30+ widgets Svelte
│   └── sdk/           # Store canvas, HyperSkill encoding
├── apps/
│   ├── flex/          # Demo en ligne (chat agent + widgets)
│   ├── viewer/        # Lecteur HyperSkill
│   ├── home/          # Landing page
│   └── showcase/      # Galerie de widgets
└── scripts/
    └── deploy.sh      # Déployer sur serveur
```

## Configuration rapide

### Exemple 1 : Boucle agent simple

```typescript
import { runAgentLoop } from '@webmcp-auto-ui/agent';
import { RemoteLLMProvider } from '@webmcp-auto-ui/agent';

const provider = new RemoteLLMProvider({
  apiKey: process.env.ANTHROPIC_API_KEY,
  model: 'claude-3-5-sonnet-20241022',
});

const result = await runAgentLoop('Affiche un graphique des ventes Q1-Q4', {
  provider,
  maxIterations: 5,
});

console.log(result.text); // Texte final de l'assistant
console.log(result.toolCalls); // Appels d'outils effectués
```

### Exemple 2 : Avec WebMCP server (widgets)

```typescript
import { createWebMcpServer } from '@webmcp-auto-ui/core';
import { autoui } from '@webmcp-auto-ui/agent';

// autoui contient déjà les 30+ widgets natifs
const layer = autoui.layer();

const result = await runAgentLoop('Affiche les stats', {
  provider,
  layers: [layer],
  maxIterations: 5,
});
```

### Exemple 3 : Avec client MCP (données)

```typescript
import { McpClient } from '@webmcp-auto-ui/core';

const mcpClient = new McpClient({
  url: 'http://localhost:3001',
  clientName: 'webmcp-auto-ui-demo',
});

await mcpClient.initialize();

const result = await runAgentLoop('Liste les recettes', {
  provider,
  client: mcpClient,
  layers: [
    // ... tes layers MCP et WebMCP
  ],
});
```

## Premier lancement

### Lancer la démo locale

```bash
# Construire tous les packages
npm run build

# Lancer l'app flex (chat agent + canvas)
cd apps/flex
npm run dev
```

L'app est accessible à `http://localhost:5173`.

### Configurer une clé API Claude (optionnel)

```bash
# Dans apps/flex/.env.local
VITE_ANTHROPIC_API_KEY=sk-ant-...
```

Sans clé, vous pouvez utiliser Gemma (in-browser WASM).
