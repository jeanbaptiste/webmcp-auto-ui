Le package `@webmcp-auto-ui/core` contient les types W3C WebMCP, le client MCP Streamable HTTP, validation JSON Schema, et helpers.

## Types W3C WebMCP

### `ModelContextTool`

Signature d'outil au standard W3C.

```ts
export interface ModelContextTool {
  name: string;                                // Unique, non-vide
  description: string;                        // Human-readable, non-vide
  inputSchema?: JsonSchema;                   // JSON Schema des paramètres
  outputSchema?: JsonSchema;                  // JSON Schema retour (optionnel)
  annotations?: ToolAnnotations;              // Hints (readOnly, destructive, etc.)
}
```

### `ToolAnnotations`

Hints pour le comportement de l'outil.

```ts
export interface ToolAnnotations {
  readOnlyHint?: boolean;        // L'outil ne modifie pas l'état
  destructiveHint?: boolean;     // L'outil est irréversible
  idempotentHint?: boolean;      // Appels multiples = même effet
  openWorldHint?: boolean;       // Interagit avec systèmes externes
  cautionHint?: boolean;         // À utiliser avec prudence
  title?: string;                // Titre human-readable (MCP SDK)
}
```

### `JsonSchema`

Union de schemas JSON (compatible JSON Schema Draft 7+).

```ts
export type JsonSchema = boolean | JsonSchemaObject;

export interface JsonSchemaObject {
  type?: JsonSchemaType | JsonSchemaType[];
  title?: string;
  description?: string;
  properties?: Record<string, JsonSchema>;
  required?: string[];
  items?: JsonSchema | JsonSchema[];
  enum?: unknown[];
  pattern?: string;
  minLength?: number;
  maxLength?: number;
  minimum?: number;
  maximum?: number;
  // ... + composition (allOf, anyOf, oneOf)
}
```

## MCP Client

### `McpClient`

Client JSON-RPC pour protocole MCP (Streamable HTTP).

```ts
import { McpClient } from '@webmcp-auto-ui/core';

const client = new McpClient(
  'http://localhost:3000/mcp',
  {
    clientName: 'MyApp',
    clientVersion: '1.0',
    timeout: 30000,
  }
);

// Initialiser
await client.initialize();

// Lister outils
const tools = await client.listTools();

// Appeler un outil
const result = await client.callTool('search_recipes', { query: 'pizza' });
```

### Types

```ts
export interface McpClientOptions {
  clientName?: string;
  clientVersion?: string;
  timeout?: number;                // ms (défaut: 30000)
  headers?: Record<string, string>;
  autoReconnect?: boolean;
  maxReconnectAttempts?: number;   // Défaut: 3
}

export interface McpInitializeResult {
  protocolVersion: string;
  capabilities: McpCapabilities;
  serverInfo: McpServerInfo;
}

export interface McpTool {
  name: string;
  description?: string;
  inputSchema?: JsonSchema;
  outputSchema?: JsonSchema;
  annotations?: ToolAnnotations;
}

export interface McpToolResult {
  content: McpToolResultContent[];
  isError?: boolean;
}

export interface McpToolResultContent {
  type: 'text' | 'image' | 'resource';
  text?: string;
  data?: string;        // base64 pour images
  mimeType?: string;
  resource?: {
    uri: string;
    mimeType?: string;
    text?: string;
    blob?: string;
  };
}
```

### Exemple

```ts
const client = new McpClient('http://localhost:3000/mcp');
await client.initialize();

try {
  const result = await client.callTool('describe_image', {
    image_path: '/path/to/image.png',
  });
  
  const textContent = result.content.find((c) => c.type === 'text');
  console.log(textContent?.text);
} catch (err) {
  console.error('Outil échoué:', err);
}
```

## Multi-MCP Client

### `McpMultiClient`

Wrapper pour gérer plusieurs serveurs MCP.

```ts
import { McpMultiClient } from '@webmcp-auto-ui/core';

const multi = new McpMultiClient();

// Ajouter serveurs
await multi.connect('recipe_server', 'http://localhost:3000/mcp');
await multi.connect('image_server', 'http://localhost:3001/mcp');

// Lister outils de tous les serveurs
const allTools = multi.listAllTools();

// Appeler un outil (autorise la route)
const result = await multi.callTool('search_recipes', { query: 'pizza' });
```

## Validation JSON Schema

### `validateJsonSchema()`

Valide une valeur contre un schema JSON.

```ts
import { validateJsonSchema } from '@webmcp-auto-ui/core';

const schema: JsonSchema = {
  type: 'object',
  properties: {
    name: { type: 'string' },
    age: { type: 'integer', minimum: 0 },
  },
  required: ['name'],
};

const result = validateJsonSchema(
  { name: 'Alice', age: 30 },
  schema
);

console.log(result.valid);   // true
console.log(result.errors);  // []
```

### Result

```ts
export interface ValidationResult {
  valid: boolean;
  errors: ValidationError[];
}

export interface ValidationError {
  path: string;           // "$.age"
  message: string;        // "Expected integer, got string"
  value?: unknown;
  expected?: unknown;
}
```

## Helpers

### `sanitizeSchema()`

Nettoie un schema JSON pour Anthropic API (supprime les champs non-supportés).

```ts
import { sanitizeSchema } from '@webmcp-auto-ui/core';

const raw = {
  type: 'object',
  $comment: 'Sera supprimé',
  properties: { name: { type: 'string' } },
};

const clean = sanitizeSchema(raw);
```


## WebMCP Server

Serveur local pour widgets + outils UI.

### `createWebMcpServer()`

```ts
import { createWebMcpServer } from '@webmcp-auto-ui/core';

const server = createWebMcpServer('myui', {
  description: 'Custom UI widgets',
});

// Enregistrer recettes (markdown frontmatter)
server.registerWidget(`
---
widget: my-widget
description: Mon widget custom
schema:
  type: object
  properties:
    title: { type: string }
---

## Usage
Appelez widget_display('my-widget', { title: 'Hello' })
`, myWidgetRenderer);

// Ajouter outils custom
server.addTool({
  name: 'custom_action',
  description: 'Action custom',
  inputSchema: { type: 'object', properties: {} },
  execute: async (params) => {
    return { ok: true };
  },
});

// Obtenir la couche pour l'agent loop
const layer = server.layer();
```

### Types

```ts
export interface WebMcpToolDef {
  name: string;
  description: string;
  inputSchema: Record<string, unknown>;
  execute: (params: Record<string, unknown>) => Promise<unknown>;
}

export interface WidgetEntry {
  name: string;
  description: string;
  inputSchema: Record<string, unknown>;
  recipe: string;            // Markdown instructions
  renderer: WidgetRenderer;
  group?: string;
  vanilla: boolean;          // true = function, false = framework component
}

export interface WebMcpServer {
  readonly name: string;
  readonly description: string;
  registerWidget(recipeMarkdown: string, renderer: WidgetRenderer): void;
  addTool(tool: WebMcpToolDef): void;
  layer(): { protocol: 'webmcp'; serverName: string; tools: WebMcpToolDef[] };
  getWidget(name: string): WidgetEntry | undefined;
  listWidgets(): WidgetEntry[];
}

export type WidgetRenderer = 
  | ((container: HTMLElement, data: Record<string, unknown>) => void | (() => void))
  | unknown;  // Svelte component, React component, etc.
```

## postMessage Bridge

Communiquer avec iframes via postMessage (WebMCP cross-window).

### `listenForAgentCalls()`

Écouter les appels d'outils depuis iframes.

```ts
import { listenForAgentCalls } from '@webmcp-auto-ui/core';

const unregister = listenForAgentCalls(async (name, args) => {
  console.log(`Outil appelé: ${name}`, args);
  return { result: 'ok' };
});

// Plus tard
unregister();
```

### `callToolViaPostMessage()`

Appeler un outil dans un parent/sibling frame.

```ts
import { callToolViaPostMessage } from '@webmcp-auto-ui/core';

const result = await callToolViaPostMessage(
  window.parent,
  'search_recipes',
  { query: 'pizza' },
  'https://example.com'
);
```

## Navigator API (W3C WebMCP)

Polyfill pour `navigator.modelContext` (enregistrement d'outils dans le navigateur).

### `initializeWebMCPPolyfill()`

Installer le polyfill WebMCP.

```ts
import { initializeWebMCPPolyfill } from '@webmcp-auto-ui/core';

initializeWebMCPPolyfill({
  forcePolyfill: false,
  allowInsecureContext: false,
  degradeGracefully: true,
  authContext: { userId: 'user-123' },
});
```

### Usage

```ts
navigator.modelContext?.registerTool({
  name: 'my_tool',
  description: 'Ma fonction custom',
  inputSchema: { type: 'object', properties: {} },
  execute: async (input) => {
    return {
      content: [{ type: 'text', text: 'Result' }],
    };
  },
  annotations: { readOnlyHint: true },
});
```
