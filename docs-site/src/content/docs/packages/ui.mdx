Le package `@webmcp-auto-ui/ui` contient tous les composants Svelte 5 pour construire une interface utilisateur agent.

## Rendu de widgets

### `<WidgetRenderer>`

Composant master pour afficher n'importe quel widget (natif, custom WebMCP, ou vanilla).

```svelte
<script>
  import { WidgetRenderer } from '@webmcp-auto-ui/ui';
  import { autoui } from '@webmcp-auto-ui/agent';

  let data = { label: 'KPI', value: '42', trend: 'up' };
</script>

<WidgetRenderer
  id="block_1"
  type="stat-card"
  {data}
  servers={[autoui]}
  oninteract={(type, action, payload) => console.log(action, payload)}
/>
```

```ts
interface Props {
  id?: string;
  type: string;
  data: Record<string, unknown>;
  servers?: WebMcpServer[];
  oninteract?: (type: string, action: string, payload: unknown) => void;
}
```

**Résolution :**
1. Cherche le widget dans `servers` (WebMCP custom)
2. Sinon cherche dans `NATIVE_MAP` (widgets Svelte natifs)
3. Sinon affiche un fallback `[type]`

### `<BlockRenderer>` (Alias)

Alias backward-compat de `<WidgetRenderer>`.

## Sélecteurs et indicateurs

### `<LLMSelector>`

Sélecteur de modèle LLM avec options locale (WASM) et remote (Anthropic).

```svelte
<script>
  import { LLMSelector } from '@webmcp-auto-ui/ui';

  let selected = 'sonnet';
</script>

<LLMSelector
  bind:value={selected}
  onchange={(model) => console.log('Model:', model)}
/>
```

**Modèles :**
- `haiku` — Claude 3 Haiku (rapide, low-cost)
- `sonnet` — Claude 3.5 Sonnet (équilibré)
- `opus` — Claude 3 Opus (puissant)
- `gemma-e2b` — Gemma 2B WASM (in-browser)
- `gemma-e4b` — Gemma 4B WASM (in-browser)

### `<GemmaLoader>`

Indicateur de chargement et statut Gemma WASM.

```svelte
<script>
  import { GemmaLoader, WasmProvider } from '@webmcp-auto-ui/ui';

  let wasmProvider = new WasmProvider({
    modelId: 'gemma-e2b',
    onStatus: (status) => { /* ... */ }
  });
</script>

<GemmaLoader
  provider={wasmProvider}
  showDownload={true}
  onReady={() => console.log('Gemma ready')}
/>
```

Affiche :
- `idle` : Pas chargé
- `loading` : % progrès (32 %, 64 %, 100 %)
- `ready` : ✓ Prêt
- `error` : ✗ Erreur + message

### `<McpStatus>`

Indicateur de connexion MCP.

```svelte
<script>
  import { McpStatus } from '@webmcp-auto-ui/ui';

  let connected = false;
  let connecting = false;
  let serverName = '';
  let toolCount = 0;
</script>

<McpStatus
  {connected}
  {connecting}
  {serverName}
  {toolCount}
  onconnect={() => { /* ... */ }}
/>
```

États :
- `● aucun MCP` (gris)
- `● connexion…` (orange)
- `● serverName · N tools` (vert)
- `● erreur: ...` (rouge)

### `<AgentProgress>`

Indicateur d'itération agent + tokens.

```svelte
<script>
  import { AgentProgress } from '@webmcp-auto-ui/ui';

  let iteration = 1;
  let maxIterations = 5;
  let totalTokens = 1234;
  let currentTool = 'search_recipes';
</script>

<AgentProgress
  {iteration}
  {maxIterations}
  {totalTokens}
  {currentTool}
  oncancel={() => console.log('Stopped')}
/>
```

Affiche barre progression + token count live.

### `<McpConnector>`

Dialogue de connexion MCP (URL input + bouton).

```svelte
<script>

  let mcpUrl = 'http://localhost:3000/mcp';
</script>

<McpConnector
  bind:url={mcpUrl}
  onconnect={(url) => connectToMcp(url)}
/>
```

## Chat et historique

### `<ChatPanel>`

Affichage historique chat avec bulle utilisateur/assistant.

```svelte
<script>
  import { ChatPanel } from '@webmcp-auto-ui/ui';
  import type { ChatFeedItem } from '@webmcp-auto-ui/ui';

  let feed: ChatFeedItem[] = [];
</script>

<ChatPanel {feed} />
```

**Types :**

```ts
export interface ChatBubble {
  role: 'user' | 'assistant';
  text: string;
  thinking?: boolean;
  timestamp?: number;
}

export interface ChatBlock {
  type: 'widget' | 'tool-call' | 'error';
  widgetType?: string;
  toolName?: string;
  data?: Record<string, unknown>;
  error?: string;
}

export interface ChatFeedItem {
  id: string;
  bubble?: ChatBubble;
  blocks?: ChatBlock[];
}
```

## Images sûres

### `<SafeImage>`

Composant image avec validation d'URL, fallback gracieux, et placeholder.

```svelte
<script>
  import { SafeImage } from '@webmcp-auto-ui/ui';
</script>

<SafeImage
  src="https://example.com/image.png"
  alt="Description"
  fallbackText="Image indisponible"
  hideOnError={false}
/>
```

**Validation :**
- Préfixes valides : `http://`, `https://`, `data:`, `/`
- URLs invalides affichent un placeholder avec icône image
- Pas de hallucination d'URLs (protection XSS)

## Message Bus (FONC)

### `bus.register()` / `bus.broadcast()` / `bus.subscribe()`

Communication inter-composants sans couplage.

```svelte
<script>
  import { bus } from '@webmcp-auto-ui/ui';

  onMount(() => {
    const unregister = bus.register('component_id', 'type', ['channel1', '*'], (msg) => {
      console.log('Message reçu:', msg);
    });
    return unregister;
  });

  function sendMessage() {
    bus.broadcast('my_id', 'channel1', { data: 'value' });
  }
</script>
```

### `bus.link()` / `bus.unlink()` / `bus.getLinks()`

Grouper les widgets et afficher les liens.

```ts
bus.link(['w1', 'w2'], 'group_name');
bus.unlink('w1', 'group_name');
console.log(bus.getLinks('w1'));  // ['group_name']
```
