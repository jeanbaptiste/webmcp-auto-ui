Le package `@webmcp-auto-ui/sdk` contient le store canvas (état widgets), l'encoding/decoding HyperSkill, et le registre de skills.

## Canvas Store

### Utilisation Svelte 5

```svelte
<script>
  import { canvas } from '@webmcp-auto-ui/sdk/canvas';  // Alias
</script>

<div>
  Widgets: {canvas.blockCount}
  {#each canvas.blocks as block (block.id)}
    <div>{block.type}</div>
  {/each}
</div>

<button onclick={() => canvas.addWidget('stat', { label: 'KPI', value: '42' })}>
  Ajouter
</button>
```

## HyperSkill Encoding/Decoding

HyperSkill est un format de sérialisation pour les projets (skills) avec compression optionnelle gzip/brotli.

### `encode()` — Créer un URL shareable

```ts
import { encodeHyperSkill } from '@webmcp-auto-ui/sdk';

const skill: HyperSkill = {
  meta: {
    title: 'Mon skill',
    version: '1.0',
    llm: 'sonnet',
    theme: { '--primary': '#ff0000' },
  },
  content: {
    blocks: [
      { type: 'stat', data: { label: 'KPI', value: '42' } },
    ],
  },
};

const url = await encodeHyperSkill(skill, window.location.href);
console.log(url);  // https://example.com/app?hs=gz.eJ8z9jAsKCqtzMgvSwEAzVoGZA...
```

**Compression :**
- Auto-compressé avec gzip si payload > 6 KB
- Requête max nginx ~8 KB
- Format : `gz.` (gzip), `br.` (brotli), ou plain base64

### `decode()` — Parser un URL/param

```ts
import { decodeHyperSkill } from '@webmcp-auto-ui/sdk';

const url = 'https://example.com/app?hs=gz.eJ8z9js...';
const skill = await decodeHyperSkill(url);

console.log(skill.meta.title);        // 'Mon skill'
console.log(skill.content.blocks);    // [...]
```

Supporte :
- URLs complètes (`https://...?hs=...`)
- Params seuls (`gz.eJ8z...`)
- Legacy base64 (fallback)

### `hash()` — Checksummer un skill

```ts
import { computeHash } from '@webmcp-auto-ui/sdk';

const h = await computeHash(
  'https://example.com/app',
  skill.content
);
console.log(h);  // 'sha256_abc123...'
```

### `diff()` — Comparer deux versions

```ts
import { diffSkills } from '@webmcp-auto-ui/sdk';

const changes = diffSkills(skill1, skill2);
console.log(changes);
// { added: [...], removed: [...], modified: [...] }
```

## HyperSkill Versioning

### `createVersion()`

Crée une entrée versionnée d'un skill.

```ts
import { createVersion } from '@webmcp-auto-ui/sdk';

const version = await createVersion(
  skill,
  'https://example.com/app',
  previousHash
);

console.log(version.hash);          // Hash SHA256
console.log(version.previousHash);  // Lien à la version antérieure
console.log(version.timestamp);     // Date création (ms)
```

## Skills Registry

Registre localStorage des skills (CRUD).

### `createSkill()`

Créer et sauvegarder un skill.

```ts
import { createSkill } from '@webmcp-auto-ui/sdk';

const skill = await createSkill({
  title: 'Mon skill',
  description: 'Description...',
  blocks: [...],
  mcpUrl: 'http://localhost:3000/mcp',
  llm: 'sonnet',
});

console.log(skill.id);  // Auto-généré UUID
```

### `getSkill()` / `listSkills()`

Récupérer un ou tous les skills.

```ts
import { getSkill, listSkills } from '@webmcp-auto-ui/sdk';

const skill = await getSkill('skill-uuid');
const all = await listSkills();  // Skill[]
```

### `updateSkill()`

Mettre à jour un skill existant.

```ts
import { updateSkill } from '@webmcp-auto-ui/sdk';

await updateSkill('skill-uuid', {
  title: 'Nouveau titre',
  blocks: [...],
});
```

### `deleteSkill()`

Supprimer un skill.

```ts
import { deleteSkill } from '@webmcp-auto-ui/sdk';

await deleteSkill('skill-uuid');
```

### `clearSkills()`

Vider tout le registre.

```ts
import { clearSkills } from '@webmcp-auto-ui/sdk';

await clearSkills();
```

### `loadSkills()` / `loadDemoSkills()`

Charger depuis file ou démo.

```ts
import { loadSkills, loadDemoSkills } from '@webmcp-auto-ui/sdk';

await loadDemoSkills();  // Charge les skills d'exemple
await loadSkills(file);  // From File input
```

### `onSkillsChange()`

Écouter les mutations du registre.

```ts
import { onSkillsChange } from '@webmcp-auto-ui/sdk';

const unsubscribe = onSkillsChange((skills) => {
  console.log('Skills changed:', skills);
});

unsubscribe();
```

## Types

```ts
export type WidgetType =
  | 'stat' | 'kv' | 'list' | 'chart' | 'alert' | 'code' | 'text' | 'actions' | 'tags'
  | 'stat-card' | 'data-table' | 'timeline' | 'profile' | 'trombinoscope' | 'json-viewer'
  | 'hemicycle' | 'chart-rich' | 'cards' | 'grid-data' | 'sankey' | 'map' | 'log'
  | 'gallery' | 'carousel' | 'd3' | 'js-sandbox'
  | (string & {});  // Accept custom widget types

export type Mode = 'auto' | 'drag' | 'chat';
export type LLMId = 'haiku' | 'sonnet' | 'opus' | 'gemma-e2b' | 'gemma-e4b';

export interface Widget {
  id: string;
  type: WidgetType;
  data: Record<string, unknown>;
}

export interface ChatMsg {
  id: string;
  role: 'user' | 'assistant' | 'system';
  content: string;
  thinking?: boolean;
}

export interface CanvasSnapshot {
  blocks: Widget[];
  mode: Mode;
  llm: LLMId;
  mcpUrl: string;
  mcpConnected: boolean;
  mcpConnecting: boolean;
  mcpName: string;
  mcpTools: McpToolInfo[];
  messages: ChatMsg[];
  generating: boolean;
  statusText: string;
  statusColor: string;
  themeOverrides: Record<string, string>;
  blockCount: number;
  isEmpty: boolean;
}

export interface HyperSkill {
  meta: HyperSkillMeta;
  content: unknown;
}

export interface HyperSkillMeta {
  title?: string;
  description?: string;
  version?: string;
  created?: string;
  mcp?: string;
  mcpName?: string;
  llm?: string;
  tags?: string[];
  theme?: Record<string, string>;
  hash?: string;
  previousHash?: string;
  chatSummary?: string;
  provenance?: { /* ... */ };
}

export interface Skill {
  id: string;
  title: string;
  description?: string;
  blocks: SkillBlock[];
  mcpUrl?: string;
  llm?: LLMId;
  theme?: Record<string, string>;
  createdAt: number;
  updatedAt: number;
}

export interface SkillBlock {
  type: WidgetType;
  data: Record<string, unknown>;
}
```

## Snapshot (React useSyncExternalStore compatible)

```ts
import { canvas } from '@webmcp-auto-ui/sdk/canvas';

// Pour frameworks sans $state (React, Vue, etc.)
function useCanvasSnapshot() {
  const [snapshot, setSnapshot] = React.useState(canvas.getSnapshot());

  React.useEffect(() => {
    return canvas.subscribe(() => {
      setSnapshot(canvas.getSnapshot());
    });
  }, []);

  return snapshot;
}
```
