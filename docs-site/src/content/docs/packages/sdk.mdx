## Vue d'ensemble

Le package `@webmcp-auto-ui/sdk` fournit la **gestion de l'état canvas** (widgets, chat, mode) et l'**infrastructure HyperSkill** pour la sérialisation et le versioning des competences.

### Composants

- **Canvas store** : état global widgets/messages avec réactivité Svelte 5
- **HyperSkill** : encode/decode/hash/diff de competences
- **Skills registry** : persistance locale et gestion CRUD
- **Demo servers** : liste des serveurs MCP de démonstration

---

## Canvas Store

Gestion réactive de l'état interface (widgets, mode, LLM, MCP, chat).

### Utilisation

```svelte
<script lang="ts">
  import { canvas } from '@webmcp-auto-ui/sdk/canvas';
</script>

<select bind:value={canvas.mode}>
  <option value="drag">Drag & Drop</option>
  <option value="chat">Chat Mode</option>
  <option value="auto">Auto</option>
</select>

<p>Mode: {canvas.mode}</p>
<p>Widgets: {canvas.blockCount}</p>
```

### API

#### Getters (réactifs — supportent `bind:`)

```ts
canvas.blocks: Widget[]              // Tous les widgets
canvas.mode: Mode                    // 'auto' | 'drag' | 'chat'
canvas.llm: LLMId                    // Modèle LLM courant
canvas.mcpUrl: string                // URL du serveur MCP
canvas.mcpConnected: boolean         // Connecté?
canvas.mcpConnecting: boolean        // Connexion en cours?
canvas.mcpName: string               // Nom du serveur MCP
canvas.mcpTools: McpToolInfo[]       // Outils MCP disponibles
canvas.messages: ChatMsg[]           // Historique chat
canvas.generating: boolean           // Génération en cours?
canvas.statusText: string            // Texte de statut MCP
canvas.statusColor: string           // CSS color class
canvas.themeOverrides: Record<string, string>
canvas.blockCount: number            // Nombre de widgets
canvas.isEmpty: boolean              // Pas de widgets?
```

#### Setters (réactifs — supportent `bind:`)

```ts
canvas.mode = 'chat'                 // Changer mode
canvas.llm = 'sonnet'                // Changer modèle LLM
canvas.mcpUrl = 'http://...'         // URL MCP
canvas.generating = true             // Marquer génération
```

#### Méthodes (mutations)

```ts
// Widgets
canvas.addWidget(type, data): Widget
canvas.removeBlock(id): void         // Supprimer widget
canvas.updateBlock(id, data): void   // Mettre à jour données
canvas.moveBlock(fromId, toId): void // Réordonner
canvas.clearBlocks(): void           // Vider canvas
canvas.setBlocks(widgets): void      // Remplacer tous

// Chat
canvas.addMsg(role, content): ChatMsg
canvas.updateMsg(id, content): void
canvas.clearMessages(): void         // Effacer chat

// MCP
canvas.setMcpConnecting(bool): void
canvas.setMcpConnected(bool, name?, tools?): void
canvas.setMcpError(message): void

// Theme
canvas.setThemeOverrides(overrides): void

// HyperSkill
canvas.buildSkillJSON(): Record<string, unknown>
canvas.buildHyperskillParam(): Promise<string>
canvas.loadFromParam(param): Promise<boolean>
canvas.loadFromUrl(url): Promise<boolean>
```

### Types

```ts
type Mode = 'auto' | 'drag' | 'chat';
type LLMId = 'haiku' | 'sonnet' | 'opus' | 'gemma-2b' | 'gemma-7b';

interface Widget {
  id: string;
  type: WidgetType;
  data: Record<string, unknown>;
}

interface ChatMsg {
  id: string;
  role: 'user' | 'assistant' | 'system';
  content: string;
  thinking?: boolean;
}

interface McpToolInfo {
  name: string;
  description: string;
  inputSchema?: Record<string, unknown>;
}
```

### Exemple complet

```svelte
<script lang="ts">
  import { canvas } from '@webmcp-auto-ui/sdk/canvas';
</script>

<!-- Sélecteur mode -->
<select bind:value={canvas.mode}>
  <option value="drag">Drag & Drop</option>
  <option value="chat">Chat</option>
  <option value="auto">Auto</option>
</select>

<!-- Affichage widgets -->
{#each canvas.blocks as widget (widget.id)}
  <div class="widget">
    <h3>{widget.type}</h3>
    <button on:click={() => canvas.removeBlock(widget.id)}>Supprimer</button>
  </div>
{/each}

<!-- Ajout widget -->
<button on:click={() => canvas.addWidget('stat', {label: 'KPI', value: '100'})}>
  + Widget
</button>

<!-- Chat -->
{#each canvas.messages as msg (msg.id)}
  <div class="message {msg.role}">
    {msg.content}
  </div>
{/each}

<input type="text" on:keydown={(e) => {
  if (e.key === 'Enter' && e.currentTarget.value) {
    canvas.addMsg('user', e.currentTarget.value);
    e.currentTarget.value = '';
  }
}} />
```

## HyperSkill Encode/Decode

Sérialisation de competences avec compression et versioning.

### encodeHyperSkill

Encode une competence en URL compressée.

```ts
export async function encodeHyperSkill(
  skill: HyperSkill,
  sourceUrl?: string
): Promise<string>
```

### Parameters

```ts
interface HyperSkill {
  meta: HyperSkillMeta;
  content: unknown;
}

interface HyperSkillMeta {
  title?: string;
  description?: string;
  version?: string;
  created?: string;
  mcp?: string;                // URL serveur MCP
  mcpName?: string;            // Nom du serveur
  llm?: string;                // Modèle LLM utilisé
  tags?: string[];
  theme?: Record<string, string>;
  hash?: string;               // Hash unique
  previousHash?: string;
  chatSummary?: string;        // Résumé conversation
  provenance?: {               // Traçabilité
    mcpServers?: string[];
    toolsUsed?: string[];
    toolCallCount?: number;
    skillsReferenced?: string[];
    llm?: string;
    exportedAt?: string;
  };
}
```

### Exemple

```ts
import { encodeHyperSkill } from '@webmcp-auto-ui/sdk';
import { canvas } from '@webmcp-auto-ui/sdk/canvas';

async function exportSkill() {
  const skill = {
    meta: {
      title: 'Ma competence',
      mcp: canvas.mcpUrl,
      llm: canvas.llm,
      theme: canvas.themeOverrides,
    },
    content: {
      blocks: canvas.blocks,
      messages: canvas.messages,
    },
  };

  const url = await encodeHyperSkill(skill);
  console.log('URL partageable:', url);
  // https://example.com?hs=H4sIAx...
}
```

### Comportement

- **Compression** : gzip automatique si > 6 KB
- **Format** : base64 URL-safe
- **Paramètre** : `hs=...` dans la query string

### decodeHyperSkill

```ts
export async function decodeHyperSkill(

```

### Exemple

```ts
import { decodeHyperSkill } from '@webmcp-auto-ui/sdk';

async function importSkill(param: string) {
  const skill = await decodeHyperSkill(param);
  console.log(skill.meta.title);
  // Restaurer
  canvas.setBlocks(skill.content.blocks ?? []);
}
```

## createVersion

Crée une version avec tracking de diff.

```ts
export async function createVersion(
  skill: HyperSkill,
  sourceUrl: string,
  previousHash?: string
): Promise<HyperSkillVersion>
```

```ts
interface HyperSkillVersion {
  hash: string;
  previousHash?: string;
  timestamp: number;
  skill: HyperSkill;
}
```

### Exemple

```ts
const v1 = await createVersion(
  { meta: {...}, content: {...} },
  'https://example.com'
);

// Plus tard, avec changements
const v2 = await createVersion(
  { meta: {...}, content: {...} },
  'https://example.com',
  v1.hash // previousHash pour tracking
);

console.log('Diff:', await diffSkills(v1.skill, v2.skill));
```

S'abonne aux changements.

```ts

  callback: (skills: Skill[]) => void
): () => void  // Retourne unsubscribe
```

### Exemple

```ts
import { createSkill, listSkills, onSkillsChange } from '@webmcp-auto-ui/sdk';
import { canvas } from '@webmcp-auto-ui/sdk/canvas';

// Sauvegarder
await createSkill('Ma skill', {
  meta: { title: 'Ma skill', mcp: canvas.mcpUrl },
  content: { blocks: canvas.blocks },
});

// Lister
const skills = await listSkills();
skills.forEach(s => console.log(s.name));

// Observer
onSkillsChange((skills) => {
  console.log('Skills mises à jour:', skills.length);
});
```

interface McpDemoServer {

  url: string;

```ts
import { MCP_DEMO_SERVERS } from '@webmcp-auto-ui/sdk';

MCP_DEMO_SERVERS.forEach(server => {
  console.log(`${server.name} (${server.region}): ${server.url}`);
});

// Connecter au premier
const server = MCP_DEMO_SERVERS[0];
canvas.mcpUrl = server.url;
```

## Framework Agnostic Canvas Store

Pour utiliser le store vanilla (sans réactivité Svelte) :

```ts
import { canvasVanilla } from '@webmcp-auto-ui/sdk/canvas';

// Accès aux mêmes méthodes
canvasVanilla.addWidget('stat', { label: 'KPI', value: '100' });
canvasVanilla.setMode('chat');

// Subscribe pattern
const unsubscribe = canvasVanilla.subscribe(() => {
  const snapshot = canvasVanilla.getSnapshot();
  console.log('État mis à jour:', snapshot);
});
```
