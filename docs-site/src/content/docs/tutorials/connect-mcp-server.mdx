Ce tutoriel montre comment connecter un **serveur MCP externe** (Model Context Protocol) et l'intégrer avec la boucle agent pour exécuter des outils à distance.

## Concepts clés

**MCP** (Model Context Protocol) est un protocole client-serveur qui permet à un LLM d'accéder à des outils distants :

<img src="/webmcp-auto-ui/diagrams/tut-connect-mcp-1.svg" alt="Agent vers MCP Client vers serveur MCP distant" />

## Étape 1 : Initialiser le client MCP

```typescript
import { McpClient } from '@webmcp-auto-ui/core';

const mcpClient = new McpClient('http://localhost:3000/mcp', {
  clientName: 'WebMCP Auto-UI',
  clientVersion: '1.0.0',
  timeout: 30000,
});

// Initialiser la connexion
await mcpClient.initialize();

// Récupérer la liste des outils disponibles
const tools = await mcpClient.listTools();
console.log('Outils disponibles :', tools);
```

## Étape 2 : Créer une couche d'outils (ToolLayer)

Une `ToolLayer` encapsule un serveur MCP avec ses outils dans le format attendu par l'agent :

```typescript
import type { ToolLayer } from '@webmcp-auto-ui/agent';
import { McpClient } from '@webmcp-auto-ui/core';

// Récupérer les outils du serveur
async function createMcpLayer(): Promise<ToolLayer> {
  const client = new McpClient('https://api.example.com/mcp');
  await client.initialize();

  const tools = await client.listTools();

  return {
    protocol: 'mcp',
    serverName: 'example-api',
    description: 'Outils d\'accès à l\'API Example',
    serverUrl: 'https://api.example.com/mcp',
    tools: tools.map(t => ({
      name: t.name,
      description: t.description ?? '',
      inputSchema: t.inputSchema,
    })),
  };
}

// Créer la couche
const exampleLayer = await createMcpLayer();
```

## Étape 3 : Intégrer avec l'agent loop

Passez le client MCP et les couches d'outils à `runAgentLoop()` :

```typescript
import { runAgentLoop } from '@webmcp-auto-ui/agent';
import { RemoteLLMProvider } from '@webmcp-auto-ui/agent';

const provider = new RemoteLLMProvider({
  proxyUrl: '/api/chat',
  model: 'sonnet',  // résout en claude-sonnet-4-6
});

const result = await runAgentLoop('Cherche les articles sur Svelte', {
  client: mcpClient,  // ← Passer le client MCP
  provider,
  layers: [exampleLayer],  // ← Passer les couches d'outils
  maxIterations: 5,
  callbacks: {
    onToolCall: (call) => {
      console.log(`Outil appelé : ${call.name}`);
      if (call.result) console.log('Résultat :', call.result);
      if (call.error) console.log('Erreur :', call.error);
    },
    onText: (text) => {
      console.log('Agent dit :', text);
    },
  },
});
```

## Exemple complet : Connecter une API Wikipedia

```typescript
import { McpClient } from '@webmcp-auto-ui/core';
import { runAgentLoop } from '@webmcp-auto-ui/agent';
import { RemoteLLMProvider } from '@webmcp-auto-ui/agent';
import type { ToolLayer } from '@webmcp-auto-ui/agent';

async function queryWikipedia(topic: string) {
  // Initialiser le client MCP pointant vers un serveur Wikipedia
  const client = new McpClient('https://mcp-wikipedia.example.com');
  await client.initialize();

  // Créer la couche d'outils
  const tools = await client.listTools();
  const wikiLayer: ToolLayer = {
    protocol: 'mcp',
    serverName: 'wikipedia',
    description: 'Accès à Wikipedia',
    tools: tools.map(t => ({
      name: t.name,
      description: t.description ?? '',
      inputSchema: t.inputSchema,
    })),
  };

  // Initialiser le LLM
  const provider = new RemoteLLMProvider({
    proxyUrl: '/api/chat',
    model: 'sonnet',
  });

  // Lancer la boucle agent
  const result = await runAgentLoop(
    `Récupère des informations détaillées sur : ${topic}`,
    {
      client,
      provider,
      layers: [wikiLayer],
      maxIterations: 5,
      callbacks: {
        onToolCall: (call) => {
          console.log(`[${call.name}]`, call.args);
        },
        onText: (text) => {
          console.log('>', text);
        },
        onDone: (metrics) => {
          console.log(`Terminé en ${metrics.totalTokens} tokens`);
        },
      },
    }
  );

  return result.text;
}

// Utilisation
const answer = await queryWikipedia('Révolution française');
console.log(answer);
```

## Étape 4 : Gérer plusieurs serveurs MCP

Vous pouvez combiner plusieurs serveurs MCP dans une seule session agent :

```typescript
async function createMultiServerSetup() {
  // Serveur 1 : API de données
  const dataClient = new McpClient('https://api.data.example.com/mcp');
  await dataClient.initialize();
  const dataTools = await dataClient.listTools();
  const dataLayer: ToolLayer = {
    protocol: 'mcp',
    serverName: 'data-api',
    description: 'Accès aux données',
    tools: dataTools.map(t => ({
      name: t.name,
      description: t.description ?? '',
      inputSchema: t.inputSchema,
    })),
  };

  // Serveur 2 : Recherche
  const searchClient = new McpClient('https://search.example.com/mcp');
  await searchClient.initialize();
  const searchTools = await searchClient.listTools();
  const searchLayer: ToolLayer = {
    protocol: 'mcp',
    serverName: 'search-engine',
    description: 'Moteur de recherche',
    tools: searchTools.map(t => ({
      name: t.name,
      description: t.description ?? '',
      inputSchema: t.inputSchema,
    })),
  };

  // Lancer l'agent avec les deux serveurs
  return await runAgentLoop(
    'Trouve et analyse les dernières tendances',
    {
      client: dataClient,  // Utilise le client primaire
      provider: new RemoteLLMProvider({ /* ... */ }),
      layers: [dataLayer, searchLayer],  // Combiner les couches
      maxIterations: 10,
    }
  );
}
```

## Gestion des erreurs

```typescript
async function safeConnectMcp(url: string) {
  try {
    const client = new McpClient(url, {
      timeout: 10000,
      autoReconnect: true,
      maxReconnectAttempts: 3,
    });

    await client.initialize();
    console.log('✓ Connexion établie');
    return client;
  } catch (error) {
    if (error instanceof Error) {
      if (error.message.includes('timeout')) {
        console.error('⚠ Serveur MCP inaccessible (timeout)');
      } else if (error.message.includes('404')) {
        console.error('⚠ Endpoint MCP non trouvé');
      } else {
        console.error('✗ Erreur de connexion :', error.message);
      }
    }
    return null;
  }
}

// Utilisation
const client = await safeConnectMcp('https://api.example.com/mcp');
if (client) {
  // Continuer avec le client
}
```

## Lazy loading des outils

Pour les serveurs avec de nombreux outils, utilisez le lazy loading avec `buildDiscoveryTools` :

```typescript
import { buildDiscoveryTools, activateServerTools } from '@webmcp-auto-ui/agent';

// Créer d'abord la couche découverte (tools légers)
const discoveryTools = buildDiscoveryTools([mcpLayer]);

// Passer au runAgentLoop
const result = await runAgentLoop('...', {
  client: mcpClient,
  provider,
  layers: [mcpLayer],
  // Les outils complets seront activés à la demande
});

// En interne, l'agent activera les outils complets quand un outil de ce serveur est appelé
```

## Pattern : Boucle agent avec MCP dans une API

```typescript
// api/agent/+server.ts
import { json } from '@sveltejs/kit';
import { runAgentLoop } from '@webmcp-auto-ui/agent';
import { RemoteLLMProvider } from '@webmcp-auto-ui/agent';
import { McpClient } from '@webmcp-auto-ui/core';

export async function POST({ request }) {
  const { message, mcpUrl } = await request.json();

  const mcpClient = new McpClient(mcpUrl);
  await mcpClient.initialize();

  const tools = await mcpClient.listTools();
  const layer = {
    protocol: 'mcp' as const,
    serverName: new URL(mcpUrl).hostname,
    description: 'MCP Server',
    tools: tools.map(t => ({
      name: t.name,
      description: t.description ?? '',
      inputSchema: t.inputSchema,
    })),
  };

  const result = await runAgentLoop(message, {
    client: mcpClient,
    provider: new RemoteLLMProvider({
      proxyUrl: '/api/chat',
      model: 'sonnet',
    }),
    layers: [layer],
    maxIterations: 5,
  });

  return json({
    text: result.text,
    toolCalls: result.toolCalls,
    metrics: result.metrics,
  });
}
```

## Intégration avec l'UI Svelte

```svelte
<script lang="ts">
  import { canvas } from '@webmcp-auto-ui/sdk/canvas';
  import { BlockRenderer } from '@webmcp-auto-ui/ui';
  import { AgentProgress } from '@webmcp-auto-ui/ui';

  let mcpUrl = 'https://api.example.com/mcp';
  let userMessage = '';
  let loading = false;

  async function runAgent() {
    if (!userMessage.trim()) return;

    loading = true;
    canvas.setGenerating(true);

    try {
      const response = await fetch('/api/agent', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ message: userMessage, mcpUrl }),
      });

      const result = await response.json();

      // Ajouter le résultat au canvas
      canvas.addMsg('user', userMessage);
      canvas.addMsg('assistant', result.text);

      // Ajouter les widgets générés
      result.toolCalls.forEach((call: any) => {
        if (call.name.includes('widget_display')) {
          canvas.addWidget('json-viewer', {
            data: call.result,
          });
        }
      });

      userMessage = '';
    } catch (error) {
      console.error('Erreur agent :', error);
      canvas.addMsg('assistant', '❌ Erreur lors de l\'exécution');
    } finally {
      loading = false;
      canvas.setGenerating(false);
    }
  }
</script>

<div class="agent-panel">
  <input
    bind:value={mcpUrl}
    placeholder="URL du serveur MCP"
    disabled={loading}
  />
  <textarea
    bind:value={userMessage}
    placeholder="Votre question..."
    disabled={loading}
  />
  <button onclick={runAgent} disabled={loading}>
    {loading ? 'En cours...' : 'Envoyer'}
  </button>

  {#if loading}
    <AgentProgress />
  {/if}

  <div class="results">
    {#each canvas.blocks as block (block.id)}
      <BlockRenderer id={block.id} type={block.type} data={block.data} />
    {/each}
  </div>
</div>

<style>
  .agent-panel {
    display: flex;
    flex-direction: column;
    gap: 1rem;
    padding: 1.5rem;
  }

  input, textarea {
    padding: 0.5rem;
    border: 1px solid #e5e7eb;
    border-radius: 0.375rem;
  }

  textarea {
    min-height: 80px;
    resize: vertical;
  }

  button:disabled {
    opacity: 0.5;
    cursor: not-allowed;
  }

  .results {
    display: grid;
    gap: 1rem;
  }
</style>
```

## Débogage

Pour inspecter les appels MCP en détail :

```typescript
// Ajouter du logging au client
const client = new McpClient(url);

// Les résultats des outils sont dans AgentResult.toolCalls
const result = await runAgentLoop(message, {
  client,
  provider,
  layers,
  callbacks: {
    onToolCall: (call) => {
      console.log('=== APPEL OUTIL ===');
      console.log('Nom :', call.name);
      console.log('Args :', JSON.stringify(call.args, null, 2));
      console.log('Résultat :', call.result);
      console.log('Durée :', call.elapsed, 'ms');
    },
  },
});
```
