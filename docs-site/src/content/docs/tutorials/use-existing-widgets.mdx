Ce tutoriel montre comment utiliser les **widgets intégrés** (stat, chart, timeline, etc.) fournis par WebMCP Auto-UI dans votre application Svelte.

## Types de widgets disponibles

WebMCP Auto-UI fournit deux catégories de widgets :

### Widgets simples
- `stat` — Statistique clé (KPI, compteur)
- `kv` — Paires clé-valeur (propriétés, métadonnées)
- `list` — Liste ordonnée d'items
- `chart` — Graphique à barres
- `alert` — Notification/alerte
- `code` — Bloc de code avec coloration
- `text` — Paragraphe de texte
- `actions` — Boutons d'action
- `tags` — Badges/tags

### Widgets riches
- `stat-card` — KPI enrichi avec delta et variante colorée
- `data-table` — Tableau triable avec colonnes
- `timeline` — Chronologie d'événements
- `profile` — Fiche profil avec avatar et stats
- `trombinoscope` — Grille de portraits
- `json-viewer` — Arbre JSON interactif
- `hemicycle` — Composition parlementaire
- `chart-rich` — Multi-series (bar, line, area, pie)
- `cards` — Grille de cartes
- `grid-data` — Grille avec highlights de cellules
- `sankey` — Diagramme de flux
- `map` — Carte Leaflet interactive
- `log` — Flux de logs
- `gallery` — Galerie d'images avec lightbox
- `carousel` — Diaporama de slides
- `d3` — Visualisations D3.js
- `js-sandbox` — Sandbox JavaScript custom

## Étape 1 : Importer BlockRenderer

```svelte
<script lang="ts">
  import { BlockRenderer } from '@webmcp-auto-ui/ui';
  import { canvas } from '@webmcp-auto-ui/sdk/canvas';
</script>
```

## Étape 2 : Créer des widgets

Utilisez le store `canvas` pour ajouter des widgets :

```svelte
<script lang="ts">
  function addStatWidget() {
    canvas.addWidget('stat', {
      label: 'Total des ventes',
      value: '€12,450',
      trend: '+12%',
      trendDir: 'up',
    });
  }

  function addChartWidget() {
    canvas.addWidget('chart', {
      title: 'Ventes mensuelles',
      bars: [
        ['Janvier', 120],
        ['Février', 190],
        ['Mars', 150],
      ],
    });
  }

  function addTableWidget() {
    canvas.addWidget('data-table', {
      title: 'Clients actifs',
      columns: [
        { key: 'name', label: 'Nom' },
        { key: 'email', label: 'Email' },
        { key: 'status', label: 'Statut' },
      ],
      rows: [
        { name: 'Alice Dupont', email: 'alice@example.com', status: 'Actif' },
        { name: 'Bob Martin', email: 'bob@example.com', status: 'Suspendu' },
        { name: 'Charlie Lenoir', email: 'charlie@example.com', status: 'Actif' },
      ],
    });
  }
</script>

<button onclick={addStatWidget}>Ajouter KPI</button>
<button onclick={addChartWidget}>Ajouter graphique</button>
<button onclick={addTableWidget}>Ajouter tableau</button>
```

## Étape 3 : Afficher les widgets

Utilisez `BlockRenderer` pour monter les widgets du canvas :

```svelte
<div class="widgets-grid">
  {#each canvas.blocks as block (block.id)}
    <BlockRenderer
      id={block.id}
      type={block.type}
      data={block.data}
    />
  {/each}
</div>

<style>
  .widgets-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 1rem;
    padding: 1rem;
  }
</style>
```

### Tableau interactif

```svelte
<script lang="ts">
  function handleRowClick(row: Record<string, unknown>) {
    console.log('Ligne cliquée :', row);
  }

  function addTable() {
    canvas.addWidget('data-table', {
      title: 'Liste des articles',
      columns: [
        { key: 'title', label: 'Titre', align: 'left' },
        { key: 'category', label: 'Catégorie', align: 'center' },
        { key: 'views', label: 'Vues', align: 'right' },
      ],
      rows: [
        { title: 'Introduction à Svelte', category: 'Tutorial', views: 1250 },
        { title: 'WebMCP Best Practices', category: 'Guide', views: 890 },
      ],
    });
  }
</script>

<div class="table-wrapper">
  {#each canvas.blocks as block (block.id)}
    {#if block.type === 'data-table'}
      <BlockRenderer
        id={block.id}
        type={block.type}
        data={block.data}
        oninteract={(type, action, payload) => {
          if (action === 'rowclick') handleRowClick(payload as Record<string, unknown>);
        }}
      />
    {:else}
      <BlockRenderer id={block.id} type={block.type} data={block.data} />
    {/if}
  {/each}
</div>
```

## Gestion des interactions

Certains widgets émettent des événements lorsque l'utilisateur intéragit avec eux :

```svelte
<script lang="ts">
  function handleWidgetInteraction(
    type: string,
    action: string,
    payload: unknown
  ) {
    console.log(`Widget ${type} → action: ${action}`, payload);

    // Exemples :
    // type='data-table', action='rowclick', payload=row object
    // type='timeline', action='eventclick', payload=event object
    // type='cards', action='cardclick', payload=card object
    // type='gallery', action='imageclick', payload={image, index}
  }
</script>

<BlockRenderer
  id={block.id}
  type={block.type}
  data={block.data}
  oninteract={handleWidgetInteraction}
/>
```

## Mises à jour dynamiques

Pour modifier un widget après sa création :

```svelte
<script lang="ts">
  function updateStat() {
    canvas.updateBlock(blockId, {
      value: '€15,000',
      trend: '+20%',
      trendDir: 'up',
    });
  }

  function clearAllWidgets() {
    canvas.clearBlocks();
  }

  function removeOneWidget(blockId: string) {
    canvas.removeBlock(blockId);
  }
</script>

<button onclick={updateStat}>Mettre à jour le KPI</button>
<button onclick={clearAllWidgets}>Tout effacer</button>
```

## Thème et personnalisation

Utilisez le `ThemeProvider` pour customiser l'apparence :

```svelte
<script lang="ts">
  import { ThemeProvider, LIGHT_TOKENS } from '@webmcp-auto-ui/ui';
</script>

<ThemeProvider theme="light" tokens={LIGHT_TOKENS}>
  <div class="widgets-grid">
    {#each canvas.blocks as block (block.id)}
      <BlockRenderer id={block.id} type={block.type} data={block.data} />
    {/each}
  </div>
</ThemeProvider>
```

## Cas d'usage : Tableau de bord dynamique

```svelte
<script lang="ts">
  import { BlockRenderer } from '@webmcp-auto-ui/ui';
  import { canvas } from '@webmcp-auto-ui/sdk/canvas';

  function buildDashboard(data: any) {
    canvas.clearBlocks();

    // KPI principal
    canvas.addWidget('stat-card', {
      label: 'Revenu mensuel',
      value: data.revenue.toLocaleString('fr-FR'),
      unit: '€',
      trend: 'up',
      variant: 'success',
    });

    // Graphique de tendance
    canvas.addWidget('chart-rich', {
      title: 'Tendance',
      type: 'line',
      labels: data.months,
      data: [{ label: 'Revenus', values: data.values }],
    });

    // Tableau des transactions
    canvas.addWidget('data-table', {
      title: 'Dernières transactions',
      columns: [
        { key: 'date', label: 'Date' },
        { key: 'amount', label: 'Montant' },
        { key: 'status', label: 'Statut' },
      ],
      rows: data.transactions,
    });
  }

  // Charger au montage
  import { onMount } from 'svelte';
  onMount(async () => {
    const data = await fetch('/api/dashboard').then(r => r.json());
    buildDashboard(data);
  });
</script>

<div class="dashboard">
  {#each canvas.blocks as block (block.id)}
    <BlockRenderer id={block.id} type={block.type} data={block.data} />
  {/each}
</div>

<style>
  .dashboard {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(350px, 1fr));
    gap: 1.5rem;
    padding: 2rem;
  }
</style>
```
