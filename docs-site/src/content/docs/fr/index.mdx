## Architecture MVC + LLM + HyperSkills

```
User prompt --> Agent Loop --> LLM (Claude/Gemma/Ollama)
                  |                    |
                  |              tool_use blocks
                  |                    |
         +--------+--------+          |
         |                  |          v
    McpLayer (DATA)    UILayer (UI)
    MCP Servers        component()
         |                  |
    Donnees brutes     Blocs rendus
         |                  |
         +--------+---------+
                  |
            BlockRenderer
                  |
            Canvas / UI
```

## Packages

| Package | Description |
|---------|-------------|
| [`@webmcp-auto-ui/core`](/webmcp-auto-ui/fr/packages/core/) | W3C WebMCP polyfill + MCP Streamable HTTP client |
| [`@webmcp-auto-ui/agent`](/webmcp-auto-ui/fr/packages/agent/) | Boucle agent, 4 providers LLM, ToolLayers, recettes, `component()` |
| [`@webmcp-auto-ui/sdk`](/webmcp-auto-ui/fr/packages/sdk/) | HyperSkill encode/decode, skills registry, canvas store |
| [`@webmcp-auto-ui/ui`](/webmcp-auto-ui/fr/packages/ui/) | 34+ composants Svelte 5, theme system, window manager |
