      type: object

## Quand utiliser
Pour des visualisations avancées non couvertes par les autres widgets — heatmaps hexagonales, graphiques radiaux, treemaps hiérarchiques, graphes de force. Choisir le preset selon le type de données.

## Comment
1. Récupérer les données via MCP
2. Choisir le preset adapté : `hex-heatmap` pour densité spatiale, `radial` pour données cycliques, `treemap` pour hiérarchies, `force` pour graphes de relations
3. Appeler `autoui_webmcp_widget_display('d3', { title: 'Répartition budget', preset: 'treemap', data: { name: 'Budget', children: [{ name: 'R&D', value: 500 }, { name: 'Marketing', value: 300 }] } })`

## Erreurs courantes
- La structure de `data` dépend du preset choisi — consulter la documentation du preset
- Ne pas confondre avec `chart-rich` qui est plus simple mais couvre bar/line/pie
