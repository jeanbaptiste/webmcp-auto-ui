---
widget: deckgl-s2
description: Render Google S2 cells. Pass S2 tokens directly.
group: deckgl
schema:
  type: object
  required: [cells]
  properties:
    cells: { type: array, description: "Array of {token: S2 cell id, value?, color?, elevation?}" }
    extruded: { type: boolean }
    elevationScale: { type: number }
    fillColor: { description: "Fallback fill color" }
    center: { type: array }
    zoom: { type: number }
    style: { type: string }
    pitch: { type: number }
---

## When to use
You already have S2-indexed data (Google's hierarchical geo-binning).

## Example
```
deckgl_webmcp_widget_display({name: "deckgl-s2", params: {
  cells: [{token:"80858004",value:10}],
  center: [-122, 37], zoom: 8
}})
```
