---
widget: mermaid-c4
description: C4 Context diagram showing people, systems, and their relationships.
schema:
  type: object
  properties:
    definition:
      type: string
      description: "Raw Mermaid C4 definition"
    title:
      type: string
    persons:
      type: array
      items:
        type: object
        required: [alias, name]
        properties:
          alias:
            type: string
          name:
            type: string
          description:
            type: string
    systems:
      type: array
      items:
        type: object
        required: [alias, name]
        properties:
          alias:
            type: string
          name:
            type: string
          description:
            type: string
          external:
            type: boolean
    relations:
      type: array
      items:
        type: object
        required: [from, to, label]
        properties:
          from:
            type: string
          to:
            type: string
          label:
            type: string
          technology:
            type: string
---
Renders a C4 Context diagram. Provide either a raw `definition` or structured `persons`, `systems`, and `relations`.

## How
1. Call `mermaid_webmcp_widget_display({name: "c4", params: {definition: "C4Context\n  Person(user, \"User\")\n  System(app, \"App\")\n  Rel(user, app, \"Uses\")"}})`

## Example
```
mermaid_webmcp_widget_display({name: "mermaid-c4", params: {definition: "C4Context\n  title E-Commerce System\n  Person(customer, \"Customer\", \"Online shopper\")\n  System(shop, \"Shop App\", \"Web storefront\")\n  System_Ext(payment, \"Stripe\", \"Payment processor\")\n  Rel(customer, shop, \"Browses and orders\")\n  Rel(shop, payment, \"Processes payments\", \"HTTPS\")"}})
```
