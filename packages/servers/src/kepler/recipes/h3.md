---
widget: kepler-h3
description: Uber H3 hexagonal cells, color/height by value.
group: kepler
schema:
  type: object
  required: [rows]
  properties:
    title: { type: string }
    rows: { type: array, description: "[{hex_id, value?}] — H3 cell ids (e.g. '8928308280fffff')" }
    enable3d: { type: boolean, description: "Extrude cells by value (default false)" }
---

## When to use
You already have aggregations indexed by H3 cell id and want a clean global hex map.

## Example
```
kepler_webmcp_widget_display({ name: "kepler-h3", params: { rows: [{hex_id: "8928308280fffff", value: 42}] } })
```
